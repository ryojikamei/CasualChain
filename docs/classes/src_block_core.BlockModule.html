verifyBlock

  • verifyBlock(core, bObj, trackingId?): Promise<gResult<number, gError>>

  • Verify a block whether any malformed have been occurred or not.

    Parameters

    • core: ccBlockType

      set ccBlockType instance

    • bObj: blockFormat

      set a block to be verified

    • Optional trackingId: string

      can be set a tracking ID to trace

    Returns Promise<gResult<number, gError>>

    returns with gResult, that is wrapped by a Promise, that contains number if it's success, and gError if it's failure. The success means the success to process the verification, so it also returns wrong block status. The status can be one of the following:

    • -2: It is a wrong data (It might not be a block)
    • -1: It is a wrong block
    • 0: It is a clean block
    • 3: It is a problematic block
